WIKI

MS-DOS is an acronym for Microsoft Disk Operating System.

Version 5.00 was first version of MS-DOS 5 and was released in June of 1991, various bugs were fixed from version 4.01 and it added more utilities for working with memory and disks, a new full-screen text editor replaced the original line editor and the Select installation program was replaced with a new Setup. The successor to this was 5.00a which was released in November of 1991 and fixed a few bugs from version 5.00. The final versions were 5.00A and 5.00A-H (please note capitalisation) which were released in Japan only due to differences in hardware. Also of note are the beta builds which were the first to be leaked of the MS-DOS line of software. The next and final official edition of MS-DOS is MS-DOS 6.
